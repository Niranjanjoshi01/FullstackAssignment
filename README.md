I)SETUP : 
1)Node.js 
2)Visual Studio Code or any IDE

II)Installation : 
Download the ZIP and extract it OR clone the repository

III)Run Backend Server : 
cd BE
npm install
node server.js
Backend will start running on:
http://localhost:5000

IV)Run Frontend Application
cd FE
npm install
npm run dev

V)Assumptions : 
1)No authentication/authorization is implemented
2)Data mocked or stored in a simple format (noDB setup)
3)Data provided is clean

VI) Improvement if more time given : 

1)Error Handling
2)Code Optimization
3)Styling & UI
4)Testing
5)Pagination
